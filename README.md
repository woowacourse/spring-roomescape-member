### 예약 조회

| HTTP Method | GET           |
|-------------|---------------|
| End Point   | /reservations |
| Status Code | 200 OK        |

#### Response Body

``` json
[
    {
        "id": long,
        "name": String,
        "date": String,
        "time": {
            "id" : long,
            "startAt" : String
        }
        "theme": {
            "id": long,
            "name": String,
            "description": String,
            "thumbnail": String
        }
    },
    {
        ...
    }
]
```

| End Point   | /reservations/available-times?date={date}&themeId={themeId} |
| Status Code | 200 OK                                                      |

#### Response Body

``` json
[
  {
    "startAt": String,
    "timeId": long,
    "alreadyBooked": boolean
  },
  {
   ...
  }
]
```

### 예약 추가

| HTTP Method | POST          |
|-------------|---------------|
| End Point   | /reservations |
| Status Code | 201 Created   |

#### Request Body

``` json
{
    "name" : String,
    "date" : String,
    "timeId" : long
    "themeId" : long
}
```

#### Response Body

``` json
{
    "id": long,
    "name": String,
    "date": String,
    "time": {
        "id" : long,
        "startAt" : String
    }
    "theme": {
        "id": long,
        "name": String,
        "description": String,
        "thumbnail": String
    }
}
```
