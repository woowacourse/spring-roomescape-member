- [x] 예약 생성 `관리자` `사용자`
- [x] 전체 예약 목록 조회 `관리자`

### 제약 및 검증
- [x] 동일 날짜 + 시간 + 테마 중복 예약 방지
- [ ] 지나간 날짜·시간에 대한 예약 생성 불가
- [ ] 예약이 존재하는 시간 삭제 불가
- [ ] 유효하지 않은 입력값 거부 (빈 이름, 잘못된 날짜 형식 등)
