## 기능 요구사항 체크리스트

### 1단계 - 테마 도메인 추가

- [x] 테마 생성 (관리자)
- [x] 테마 삭제 (관리자)

#### API 명세

`GET /theme` 테마 리스트 조회

`POST /admin/themes` 테마 생성 (관리자)

`DELETE /admin/themes/{id}` 테마 삭제 (관리자)

### 2단계 - 사용자 예약

- [ ] 날짜와 테마로 필터링하여 예약 가능한 시간 목록 조회
- [ ] 같은 테마에 날짜와 시간이 같은 예약이 존재하는 경우 예약이 불가능하도록 처리

#### API 명세

`GET /themes/{themeId}/times?date={date}` 예약 가능한 시간 목록 조회
