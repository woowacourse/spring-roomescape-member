### API
#### 예약 가능 시간 조회 API
**request**
GET /times/memeber?date=2024-05-02&themeID=1 HTTP/1.1

**response**
HTTP/1.1 200
Content-Type: application/json

[
{
"id": 1,
"startAt": "19:25:00",
"alreadyBooked": false
}
]

GET /themes/ranking HTTP/1.1

HTTP/1.1 200
Content-Type: application/json

[
{
"id": 1,
"name": "교실방탈출",
"description": "교실테마입니다.",
"thumbnail": "class.jpg"
}
]
