## 1. 기능 요구 사항
### 1. 예약
1. 예약자와 관리자는 예약을 생성한다.
2. 모든 예약을 조회한다. 
3. 특정 예약을 삭제한다.

### 2. 타임대
1. 타임대를 추가한다.
2. 전체 타임과 예약 가능 상태를 조회한다.
3. 특정 타임대를 삭제한다.

### 3. 테마
1. 테마를 생성한다.
2. 특정 테마를 삭제한다.
3. 테마들을 조회한다.(옵션: 날짜)
4. 1주 인기 테마들을 조회한다
