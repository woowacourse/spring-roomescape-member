사용자 예약 API

  • GET /reservations: 지정한 사용자 이름으로 내 예약 현황 조회
  • POST /reservations: 지정한 사용자 이름·날짜·시간 ID로 예약 생성
  • DELETE /reservations/{id}: 지정한 사용자 이름으로 본인 예약 취소

테마/시간 API

  • GET /themes: 예약 페이지 테마 목록
  • GET /themes/popular: 인기 테마
  • GET /themes/{themeId}/available-times: 날짜별 가능 시간

예외 안내

  • 400: 입력값/날짜/JSON 형식 오류
  • 401/403: 인증 헤더 누락 또는 본인 예약 아님
  • 404/409: 대상 없음 또는 중복·제약 조건 위반
