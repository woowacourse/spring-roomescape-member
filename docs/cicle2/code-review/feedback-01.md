### 코드 가독성

> 어노테이션/필드의 줄바꿈

- 의도/기준/생각
    - 하드 랩을 넘어가면 해당 줄의 말단을 각 줄마다 개행
    - 넘어가지 않더라도 열거의 필요성이 크다면 개행
    - 어노테이션 길이가 늘어난다면 필드와 어노테이션을 개행하는 것도 효과적
