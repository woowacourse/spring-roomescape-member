# 2. 사용자 흐름 정의

## 1. 사용자 (Client) 예약하기
- **액터**: 사용자
- **목표**: 원하는 날짜와 테마의 빈 시간을 찾아 방탈출 예약을 완료한다.
- **성공 흐름**:
  1. 사용자가 메인 페이지에 진입한다.
  2. 원하는 방탈출 테마를 선택한다. (인기 테마 또는 전체 테마 목록에서)
  3. 원하는 날짜를 선택한다.
  4. 해당 날짜/테마의 예약 가능한 시간 목록 중 하나를 선택한다.
  5. 본인의 이름을 입력하고 예약 버튼을 클릭한다.
  6. 예약이 완료되었다는 안내와 함께 내 예약 조회 화면으로 이동한다.
- **대체/실패 흐름**:
  - 선택한 시간에 이미 다른 예약이 차버린 경우: 예약 실패 알림을 띄우고 시간을 다시 선택하게 한다.
  - 이름을 입력하지 않은 경우: 이름 입력 안내 문구를 띄운다.

## 2. 사용자 (Client) 예약 조회/변경/취소
- **액터**: 사용자
- **목표**: 내 예약을 확인하고, 일정을 변경하거나 취소한다.
- **성공 흐름**:
  1. 사용자가 '내 예약 조회' 페이지에 진입한다.
  2. 본인의 이름을 입력하고 조회 버튼을 클릭한다.
  3. 예약 내역 목록을 확인한다.
  4. 특정 예약의 '일정 변경'을 클릭한다.
  5. 변경할 날짜와 시간을 선택하고 수정 버튼을 클릭한다.
  6. 일정이 성공적으로 변경된다.
- **대체/실패 흐름**:
  - (취소 흐름) 예약의 '예약 취소'를 클릭하면 예약을 즉시 취소한다.
  - (실패 흐름) 입력한 이름으로 조회된 예약이 없는 경우: "예약 내역이 없습니다"라는 안내를 띄운다.

## 3. 관리자 (Admin) 예약 시간 관리
- **액터**: 관리자
- **목표**: 예약을 받을 시간(슬롯)을 추가하거나 삭제한다.
- **성공 흐름**:
  1. 관리자가 '예약 시간 관리' 페이지에 진입한다.
  2. 텍스트 필드에 시간(ex: `10:00`)을 입력하고 추가 버튼을 클릭한다.
  3. 하단 시간 목록에 새 시간이 추가된다.
- **대체/실패 흐름**:
  - 기존 시간을 삭제하고자 할 때 '삭제' 버튼을 클릭하면 즉시 삭제된다.
  - 이미 존재하는 시간을 추가하려는 경우: 에러 알림을 띄운다.

## 4. 관리자 (Admin) 테마 관리
- **액터**: 관리자
- **목표**: 방탈출 테마를 새로 등록하거나 삭제한다.
- **성공 흐름**:
  1. 관리자가 '테마 관리' 페이지에 진입한다.
  2. 테마 이름, 설명, 썸네일 이미지(URL 등)를 입력하고 추가 버튼을 클릭한다.
  3. 하단 테마 목록에 새 테마가 추가된다.
- **대체/실패 흐름**:
  - 기존 테마를 삭제하고자 할 때 '삭제' 버튼을 클릭하면 즉시 삭제된다.
  - 필수 입력값(이름, 설명 등)을 누락한 경우: 입력 안내 문구를 띄운다.

## 5. 관리자 (Admin) 예약 관리
- **액터**: 관리자
- **목표**: 전체 사용자의 예약 내역을 조회하고 필요시 예약을 강제 취소한다.
- **성공 흐름**:
  1. 관리자가 '예약 관리' 페이지에 진입한다.
  2. 현재 등록된 모든 예약 내역을 페이징 단위로 확인한다.
  3. 악성 예약 등 관리가 필요한 예약의 '취소(삭제)' 버튼을 클릭한다.
  4. 목록에서 해당 예약이 삭제(취소)된다.
- **대체/실패 흐름**:
  - 목록에 예약이 아예 없는 경우: 빈 리스트를 표시한다.
