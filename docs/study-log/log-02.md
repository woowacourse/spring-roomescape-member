### 4. AI 피드백
- 자신의 학습 전략에 대해 AI 학습 전문가에게 피드백을 요청하고, 유용했던 제안 1가지 이상 기록
  * 제안 1: Depth 기준 모호함 → 새로운 개념 등장 여부로 판단하기
    * 새로운 개념이 등장할 때마다 Depth +1
    * 같은 개념에 대한 추가 질문은 Depth 유지

  * 제안 2: 현재 Depth 추적 어려움 → 학습 스택을 시각적으로 기록하기
    * 브라우저의 탭처럼, 학습 도중 파생된 개념을 스택(stack) 형태로 메모해두는 습관을 권장한다.
   
  > [현재 구현 목표] Reservation 저장 기능
  > → JdbcTemplate.update() (새로운 개념 → Depth 1)
  > → update() 파라미터가 뭔지, 반환값이 뭔지 (같은 개념 → Depth 1 유지)
  > → KeyHolder가 왜 필요한지 (새로운 개념 → Depth 2)
  > → KeyHolder 내부 동작 (같은 개념 → Depth 2 유지)
  > → GeneratedKeyHolder와의 차이 (새로운 개념 → Depth 3, 마지막 허용 단계)
  > → (여기서 더 들어가면 Depth 초과)

### 5. 다음 타임에 바꿀 것

- 바꿀 것과 그 이유
