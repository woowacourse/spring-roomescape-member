# 방탈출 예약 관리

## 학습법 변천 📊

### 1️⃣ V1

1. 혼자 고민하기
2. 토론 & 스터디
3. 리뷰어 DM

문제점:

1. 완벽하게 하려고 했던 것
2. 혼자서 고민하려고 했던 것
3. 과몰입해서 깊게 들어가거나 중간에 딴 길로 새게 됨

### 2️⃣ V2

1. 스텝을 작게 쪼개고 글로 목표를 적어서 보이는 곳에 두기
2. 각 스텝마다 데드라인을 잡아두기

### 3️⃣ V3

1. 스텝을 작게 쪼개고, 각 스텝의 목표를 글로 적는다.
2. 각 스텝마다 데드라인을 잡는다.
3. 스텝이 끝날 때마다 30초 설명을 쓴다.
