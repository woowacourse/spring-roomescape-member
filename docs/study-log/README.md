## 학습법

### 학습법_v1
1. 개념 및 우선순위 식별
    - 학습 전, AI에게 현재 미션 상황에서 필요한 개념이 무엇인지 질문하여 학습할 키워드와 우선순위를 정한다.
2. 자료 탐색 및 학습
    - AI에게 해당 개념과 관련된 공식 문서 링크를 요청하여 학습한다.
    - 공식 문서가 없거나 부족할 경우 기술 블로그, 책 등 공신력 있는 자료를 찾아본다.
    - 적절한 자료를 찾기 어렵다면 AI에게 질문하여 내용을 파악한다.
3. 질의 및 교차 검증
    - 학습 중 이해가 되지 않는 부분은 AI에게 질문한다.
    - AI의 답변이 정확한지 외부 자료를 통해 일일이 교차 검증한다.
4. 반복 및 학습 종료
    - 스스로 이해되었다고 느낄 때까지 위 과정을 반복한다.

### 학습법_v3
1. 개념 및 우선순위 식별

2. 자료 탐색 및 학습

3. 질의 및 교차 검증
    - 학습 중 이해가 되지 않는 부분은 AI에게 질문한다.
    - AI 답변의 적절성을 검증하되, 검증 시간은 30분으로 제한한다.
    - 시간이 지났음에도 이해되지 않은 부분은 기록한 후 넘어간다.
    - 학습 깊이 제한:
        - Depth 기준 : 원래 개념에서 새 질문이 파생될 때마다 Depth 1씩 증가한다.
        - 배경 지식(상위): 이해를 돕기 위한 직전 개념은 Depth 1까지만 확인한다.
        - 대상 지식(연관): 현재 학습할 개념과 그로부터 이어지는 흐름은 Depth 3까지로 제한한다.
    - Depth 제한 도달 시 이해 판정 단계로 넘어간다.
4. 이해 판정
   - 아래 두 질문에 답하며 이해를 판정한다.
       - 이게 무엇인가? → 요약
       - 이게 왜 필요한가? → Why
   - 두 질문에 답할 수 있으면 이해한 것으로 판정, 없으면 이해되지 않은 부분으로 기록한다.

### 학습법_v4
1. 개념 및 우선순위 식별
   - 학습 전, 현재 상황에서 필요한 개념이 무엇인지 생각한다.
   - AI에게 현재 상황에 필요한 개념을 질문하여 학습 키워드와 우선순위를 보완한다.
   - 학습할 개념이 기존 개념과 연결되는 경우, 구현 전에 개념 간 관계를 파악한다.
2. 자료 탐색 및 학습
   - AI에게 해당 개념과 관련된 공식 문서 링크를 요청하여 학습한다.
   - 공식 문서가 없거나 부족할 경우 기술 블로그, 책 등 공신력 있는 자료를 찾아본다.
   - 적절한 자료를 찾기 어렵다면 AI에게 질문하여 내용을 파악한다.
   - 20분이 지나면 반드시 멈추고 전환 단계로 넘어간다.
3. 전환
   - 학습한 내용을 바탕으로 아래 중 하나를 진행한다.
      - 기록: 학습한 내용을 정리하고 이해되지 않은 부분을 기록한다.
      - 구현: 학습한 개념을 바탕으로 구현을 시도한다.
      - AI 토론: 이해가 불분명한 부분을 AI와 토론한다.
   - 전환이 끝나면 이해 판정으로 넘어간다.
4. 이해 판정
   - 아래 두 질문에 답하며 이해를 판정한다.
      - 이게 무엇인가? → 요약
      - 이게 왜 필요한가? → Why
   - 판정 성공: 다음으로 넘어간다.
   - 판정 실패: 이해되지 않은 부분을 기록하고 1단계부터 사이클을 반복한다.
